## Description

Task Manager exercise build with NestJS with:

- Get, Post, Delete and Patch endpoints to add, create delete and update tasks.
- Postgre data persistance
- Authentication using hashed passwords and JSON Web Tokens

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
